01. Project Description

What is Homework Tracker?

This is a smart web application designed specifically for teenagers (10-17 years old) to help organize the learning process and turn the chaos of deadlines into a clear action plan.

Problem & Importance

Many students forget about homework or do it at the last minute. This causes stress and lower grades. Our product solves this problem by helping students develop responsibility.

Why it's Unique?

Unlike complex paid organizers, our application is completely free, requires no installation, and focuses only on what is most important for a student.

Target Audience

Teenagers 10-17 years old who value simplicity and want to access their tasks from both their computer and smartphone without annoying subscriptions.

02. Requirements & Tech Stack

Functional Requirements

  • User registration and secure login via password.
  • Creating tasks specifying subject, date, and description.
  • Automatic status system based on deadlines.
  • In-app smart notifications specifically for tomorrow's deadlines.
  • Storing user data in the browser locally.

Technical Requirements

  • Languages: HTML5, CSS3, JavaScript (Vanilla).
  • Storage: LocalStorage API for secure client-side saving.
  • Hosting: GitHub Pages.
  • Version Control: GitHub Repository.

03. Prototype & Visual Identity
